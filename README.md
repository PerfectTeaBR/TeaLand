# 💫 TeaLand - Algumas coisinhas para o universo da PerfectTea!

Eu já estava planejando fazer a TeaLand á muito e muito tempo, então, depois de muito planejamento, ela está aqui!

O Objetivo é ser uma API para a **LuaBot**, retornando modelos usando um próprio servidor com **FastAPI**.

## 💻 Tecnologias Utilizadas

| Nome     | Categoria                 |
| -------- | ------------------------- |
| Python   | Linguagem Principal       |
| FastAPI  | Framework para fazer APIs |
| Pydantic | Para os Modelos usados    |

## Instalando as libs

```bash
pip install fastapi
pip install uvicorn
pip install pydantic
```
